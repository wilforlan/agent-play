# Talk Timer Challenge

**Cabinet:** Comms Booth (`talk-timer`)

## Play loop

Hold **Space** (or the on-screen transmit control) to land the needle in the green window. Three rounds with shrinking windows. No real talk billing.

## PU rules

| Event | PU |
|-------|-----|
| Success per round | +5 |
| Fail per round | −2 |
